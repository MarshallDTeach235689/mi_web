¿Qué es VSC? - Concepto

Visual Studio Code, es importante saber qué es un IDE. Sus siglas en inglés “Integrated Development Environment “ hacen referencia a un entorno de desarrollo integrado​​ o interactivo. Visual Studio es una aplicación informática cuya finalidad es simplemente hacer la vida de los desarrolladores y programadores mucho más fácil, al ofrecer servicios integrales que facilitan desarrollar el software de manera mucho más sencilla.

Fuente: Innova consulting

Proceso de instalación

Para la instalación de Visual Studio Code en Windows sera la siguiente:

  1. Paso 1:

    2. Seleccionar y descargar la versión que corresponda a su sistema operativo, por ejemplo: Windows, User Installer, 64 bit una vez se descargue el archivo, abrirlo para iniciar el proceso de instalación.

  2. Paso 2:

    3. Aceptar los términos del acuerdo y presionar el botón “Siguiente>”

  3. Paso 3:

    4. Seleccionar la carpeta donde desea realizar la instalación o presionar el botón “Siguiente>” para instalar en la carpeta por defecto (recomendado).

  4. Paso 4:

    5. Seleccionar la carpeta del menú de inicio para colocar los accesos directos del programa o presione el botón “Siguiente>” para que se cree una carpeta por defecto (recomendado).

  5. Paso 5:

    6. Seleccionar todas las casillas de verificación y presionar el botón “Siguiente>”

  6. Paso 6:

    7. Presionar el botón “Instalar” para continuar con la instalación.

  7. Paso 7:

    8. El proceso de instalación avanza durante algunos segundos.

  8. Paso 8:

    9. Presione el botón “Finalizar” para terminar el proceso de instalación y seguido a esto se abrirá el programa Visual Studio Code.
